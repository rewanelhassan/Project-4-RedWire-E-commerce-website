# Red Wire | E-Commerce Electronics Store

A modern, high-performance e-commerce platform built with **HTML5, CSS3, Bootstrap 5, and Vanilla JavaScript**.

## 🚀 Features
* **Full Responsive Design:** Seamless experience on Mobiles, Tablets, and Desktops.
* **Dynamic Search:** Filter products by Name or Category using JavaScript.
* **Shopping Cart:** Interactive cart functionality with real-time updates.
* **Modern UI:** Bold and energetic design inspired by the "Red Wire" brand.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Bootstrap 5.
* **Scripting:** Vanilla JavaScript (DOM Manipulation, LocalStorage).
* **Icons:** FontAwesome.
